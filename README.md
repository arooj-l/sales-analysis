# Supermarket Sales Analysis with Pandas

##  Overview
This project is a beginner-friendly introduction to the **Pandas** library in Python. It demonstrates how to use **pandas module functions** and **DataFrame methods** to load, explore, analyze, and manipulate real-world sales data.

The project simulates a supermarket sales dataset and performs various data analysis tasks such as calculating revenue, identifying top-selling products, and summarizing category performance.

---

##  Objectives
- Understand the difference between **pandas module functions** and **DataFrame methods**.
- Learn how to create and manipulate **DataFrames**.
- Perform real-world **data analysis** tasks.
- Practice **data cleaning**, **filtering**, **grouping**, and **sorting**.
- Build a strong foundation for data science and analytics.

---

##  Technologies Used
- **Python 3**
- **Pandas**
